# TCU Rebuild

This app is the new umbrella-site rebuild for The Carbon Underground.

## Scope

- Lead with `The Other Half / Two Halves`
- Make Edison the flagship announcement and proof point
- Keep v1 focused on Edison, Adopt-A-Meter, and Education / Thought Leadership
- Keep the stats microsite separate and linked as an evidence hub

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Related Folders

- `../REFERENCE_LIBRARY` contains the organized discovery inputs
- `../TCUSoilStats` remains the separate evidence microsite
- `./logs` holds archived local dev and preview session logs when the app root gets swept
